Um pouco sobre a história do HTML

(Voltar)

O HTML foi inventado em 1990 por um cientista chamado Tim Berners-Lee. A finalidade inicial era a de tornar possível o acesso e a troca de informações e de documentação de pesquisas, entre cientistas de diferentes universidades. O projeto inicial tornou-se um sucesso jamais imaginado por Tim Berners-Lee.

"A Web não apenas conecta máquinas, conecta as pessoas."

Tim Berners-Lee

Como funciona

(Voltar)

O HTML é uma linguagem de marcação. Estas linguagens são constituídas de códigos que delimitam conteúdos específicos, segundo uma sintaxe própria. O HTML tem códigos para criar paginas na web, estes códigos que definem o tipo de letra, qual o tamanho, cor, espaçamento, e vários outros aspectos do site.

O que são tags

(Voltar)

Todas as marcações são realizadas pelas tags que em geral são estruturas de linguagem de marcação que contém instruções, as tags também são responsáveis por criar elementos nas páginas web com a função de organizar, identificar e limitar o conteúdo de uma página. As tags são muito importantes e fundamentais para o desenvolvimento WEB, e usa-las de forma correta vai agregar bastante para o seu site.

Tipos de tags

(Voltar)

head: Providencia informações gerais (metadados) sobre o documento, incluindo seu título e links para scripts e folhas de estilo;

href: Vincula a URL do documento a algo;

heading tags (h1, h2, h3...): São recursos de programação HTML utilizados para destacar títulos e subtítulos de uma página;

body: Representa o conteúdo de um documento HTML;

target: Identifica se o link deve ser aberto na mesma janela, em nova aba ou nova janela;

img: Permite inserir imagens na página da web;

u/i: Permite sublinhar e tornar o texto itálico;

ol: Cria listas ordenadas. Deve ser acompanhado de start, para indicar o começo da lista e type, para apontar o tipo de caractere que será ordenado;

ul: Cria listas não ordenadas. Precisa do type para apontar o tipo de item que fará parte da lista;

li: Define cada item da lista

strong: Forma o texto com negrito;

blockquote: Permite inserir uma citação ao texto;

p: Define o começo e o fim de um parágrafo. Pode ser utilizado ao lado de align para indicar o alinhamento do trecho;

a: Cria um link e atribui elementos a ele;

br: Define uma quebrar de linha (Pula uma linha);

hr: Representa uma quebra temática entre elementos de nível de parágrafo.
